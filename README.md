# Sistema de Marmitas

Este programa simula o atendimento de uma empresa de marmitas, permitindo ao cliente escolher opções do cardápio e calcular o valor final do pedido.

## Funcionalidades
- Catálogo de marmitas
- Entrada do pedido do cliente
- Cálculo do valor total
- Exibição do pedido finalizado

## Tecnologias
- Python 3

## Como executar
```bash
python sistema_marmitas.py
